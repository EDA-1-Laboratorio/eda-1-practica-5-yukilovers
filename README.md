# Práctica 5. Pilas
 ## Ejercicio 1. La calculadora polaca
El script del ejercicio 1 contiene el código que implementa el cálculo de operaciones aritméticas mediante el uso de pilas con notación polaca. En el encabezado del código, se invoca a "pila.h", en donde se enuncia una serie de funciones básicas de las pilas, pero no se definen. El objetivo del ejercicio es terminar la definición de esas funciones. Ocupen las descripciones que se incluyen en el código para identificar las partes que se requiere completar.

 ## Ejercicio 2. La pila de palíndromos
 En el script del ejercicio 2, se deben de completar las funciones señaladas en el código. El objetivo es identifcar si la cadena de entrada es un palíncdromo.
